# HUB SENNAF – MCH  
Auditoría de sistema institucional (QA + datos reales)

**Estado:** En curso / Iterativo  
**Contexto:** Sistema institucional en entorno de pruebas

## Descripción general
Este caso documenta una auditoría sistémica del HUB SENNAF (MCH), con foco en testing funcional (QA), validación con datos técnicos reales, observaciones de usabilidad/UX y detección de barreras básicas de accesibilidad.

El trabajo se desarrolló a partir de una metodología multicapa que integra sanity tests, flujos funcionales, pruebas negativas, testing con datos reales y análisis de desajustes entre inventarios técnicos y el modelo de datos del sistema.

## Alcance
- QA funcional (sanity tests, flujo feliz y escenarios negativos)
- Pruebas con datos técnicos reales (inventario de servidores)
- Observaciones de usabilidad y experiencia de usuario
- Diagnóstico básico de accesibilidad
- Análisis de riesgos operativos e institucionales

## Eje del caso
La auditoría pone en evidencia cómo la falta de alineación entre datos reales y modelo de sistema, la ausencia de feedback al usuario y los flujos poco claros pueden generar riesgos operativos, administrativos e institucionales en sistemas públicos.
